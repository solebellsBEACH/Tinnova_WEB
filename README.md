## Instruções para rodar o projeto em desenvolvimento
#### 1. `git clone https://github.com/solebellsBEACH/Tinnova_WEB.git`  para clonar o repositório

#### 2. `cd Tinnova_WEB/`

#### 3. `yarn` para rodar as dependências

#### 4. `yarn dev`

## Testes Automatizados
